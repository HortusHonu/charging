# Coding Challenge for Brilliant Chemistry

#### Ruby Version:
* 2.5.1

#### Rails Version:
* 5.2.2

#### Database
* Postgresql

### Test Suite:
* Rspec 3.8.0 and Capybara 

#### Notes:

* Setup app with:
~~~
bundle install
~~~

* Configure database with 
~~~
rake db:create rake db:migrate rake db:seed
~~~

* Run tests with
~~~
bundle exec rspec
~~~
